## 소개
핵심단어를 드래그 함으로써 가독성을 높이다.  
이미 읽었던 포스팅을 다시 읽을 때,   
필요한 부분만 다시 읽을 수 있도록,  
드래그함으로써 텍스트에 하이라이팅효과를 부여한다.  

## 단축키 
shift + V ->  On/Off,  
shift + X -> 모두 지우기,  
shift + Z -> 행동취소(이전 상태로 되돌리기)  

@ 단어에 특화,  
@ 문장과 문단의 하이라이팅은 기대하기 힘듬.
